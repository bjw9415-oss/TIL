## 오늘 배운 것들

#### 부트스트랩(Bootstrap)

```
웹 페이지를 쉽고 빠르게 만들 수 있도록 도와주는 CSS+ JavaScript 기반의 포론트엔드 프레임 워크
```

- 하나의 거대한 CSS파일 속에 이미 버튼, 카드, 그리드, 여백, 색상 등의 스타일이 정의 되어 있다
- HTML 요소에 클래스를 추가하는 것만으로도 완성도 높은 화면 구성 가능하다~

- [공식 홈페이지](https://getbootstrap.com/)

#### 사용방법

1. head 아래를 안에 넣어준다
   link href="https://cdn.jsdelivr.net/npm/bootstrap@5/dist/css/bootstrap.min.css" rel="stylesheet"
   를 넣는다

2. body 끝에 넣어준다
   script src="https://cdn.jsdelivr.net/npm/bootstrap@5/dist/js/bootstrap.bundle.min.js"></script>

- **라이브러리(Library)**

→ 특정 기능을 수행하는 도구 모음.

개발자가 필요할 때 **직접 호출해서 사용하는 보조도구**이다.

예) jQuery, Chart.js, Lodash

- **프레임워크(Framework)**

→ 애플리케이션의 구조와 흐름을 미리 정해둔 **골격**이다.

개발자는 그 틀 안에서 코드를 작성하며, **프레임워크가 제어권을 가진다.**

예) Bootstrap, React, Angular

> 부트스트랩은 프레임워크다.
